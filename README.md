# 🚚 MoveMate

A Flutter-based logistics and transportation booking application inspired by Porter.

## Features

* 🔐 User Login
* 📦 Service Selection
* 🚛 Vehicle Selection
* 📍 Pickup & Drop Locations
* 💰 Dynamic Fare Display
* ✅ Booking Confirmation

## Tech Stack

* Flutter
* Dart
* Material Design

## Screens

* Welcome Screen
* Login Screen
* Home Dashboard
* Service Selection
* Vehicle Selection
* Location Details
* Fare Summary
* Booking Confirmation
